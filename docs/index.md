## Installation

1. Download add-on (zipped) from the top.
2. In blender, go to : Edit -> Preferences -> Add-ons -> Install.
3. Select the downloaded file and you're good to go.

## Features

* Floorplans
* Floors (slabs and walls)
* Doors
* Windows
* Multigroup (door-window combinations)
* Roof
* Stairs
* Balcony
